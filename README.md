

## Analisis Pola Pembelian Konsumen Menggunakan Algoritma Apriori pada Dataset Online Retail

Proyek ini bertujuan untuk menemukan pola pembelian konsumen menggunakan metode Association Rule Mining dengan algoritma Apriori pada dataset Online Retail. Hasil analisis digunakan untuk menghasilkan rekomendasi strategi bisnis seperti product bundling, cross-selling, dan sistem rekomendasi produk.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Mlxtend

## Metode
- CRISP-DM
- Association Rule Mining
- Apriori Algorithm

## Hasil
- Frequent Itemsets : 1102
- Association Rules (Lift > 1) : 174
- Average Confidence : 61.90%
- Average Lift Ratio : 21.35
